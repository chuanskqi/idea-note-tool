### IDEA代码书签工具 
项目由来:
idea社区版,Bookmarks工具有一些bug,例如切换项目分支时,书签会全部消失。 
鉴于看源码提pr坡度较高,功能较为简单,就自己练手先写个插件实现吧

### 开发历程
耗时2个周末,主要花费时间再阅读idea插件开发文档,idea封装的接口挺方便,代码实现简单,2022.06.12
